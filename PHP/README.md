# PHP

PHP 는 1955년 6월 8일에 젠드 테크놀로지스가 발표한 프로그래밍 언어이다.

### PHP 설치하는법

<br>

+ #### Windodws 
  
    [PHP Install Link](https://windows.php.net/download/)

+ #### Ubuntu Linux

    ```bash
    ~$apt install php
    ```

+ #### CentOS Linux

    ```bash
    ~$yum install php -y
    ```

+ #### alpine Linux

    ```bash
    ~$apk add php
    ```
